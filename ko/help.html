CarryOver 사용 방법

CarryOver는 링크, 텍스트, 관심 있는 순간을 지금 저장하고 나중에 알맞은 기기에서 검토하고 실행하도록 도와줍니다.

콘텐츠 저장

아무 앱이나 열기

Safari, 메모, 소셜 앱처럼 텍스트나 URL을 공유할 수 있는 앱에서 시작하세요.

텍스트 또는 URL 선택

나중에 검토할 내용을 선택합니다.

공유 사용

시스템 공유 버튼을 누르고 CarryOver를 선택하세요. 보이지 않으면 더보기에서 활성화합니다.

저장 확인

항목이 최근 목록에 추가되고 iCloud 동기화를 준비합니다.

목록 관리

최근 저장 항목 보기

CarryOver를 열어 최근 저장한 항목을 확인하세요.

감지된 링크 열기

URL이 포함된 항목은 브라우저에서 이어 볼 수 있습니다.

항목 삭제

더 이상 필요 없는 항목을 목록에서 삭제하세요.

기록 사용

처리한 내용을 다시 확인할 때 기록에서 찾을 수 있습니다.

동기화와 문제 해결

같은 iCloud 계정 사용

같은 Apple ID로 로그인하고 iCloud를 사용할 수 있어야 합니다.

동기화 다시 시도

동기화 문제가 표시되면 상태 아이콘으로 다시 시도하세요.

iCloud 저장 공간 확인

다른 기기에 바로 나타나지 않으면 iCloud 공간이나 백그라운드 업로드 상태를 확인하세요.

자주 묻는 질문

공유 시트에 CarryOver가 보이지 않아요.

공유 시트의 더보기에서 CarryOver를 활성화하고 필요하면 순서를 조정하세요.

다른 기기에 바로 동기화되지 않아요.

네트워크, 저전력 모드, iCloud 저장 공간 부족으로 지연될 수 있습니다.

CarryOver 계정이 필요한가요?

아니요. 가능한 경우 Apple의 iCloud 서비스로 동기화합니다.
